# kudismart

A smart personal finance management platform that helps individuals gain visibility into their money, track expenses, manage bills, monitor subscriptions, and make better financial decisions through actionable insights **


## Overview
Many people earn money, pay bills, subscribe to services, and spend daily, yet struggle to clearly understand their financial habits.

KudiSmart was built to solve this problem by providing a centralized platform where users can:

**Track daily expenses**
**Monitor bills and subscriptions**
**View spending trends**
**Analyze financial habits**
**Receive intelligent financial insights**
**Manage wallet transactions**
**Improve budgeting decisions**

*** It is still in MVP phase
